새 댓글 작성 추가

  • 새로운 댓글을 작성하는 기능을 추가하겠습니다.
  • 앞에서 작성한 datt.html 을 보면 url 'posts:createdatt' post.id으로 createdatt의 url이 되어있는것을 확인할 수 있습니다.
  • 이러한 url의 결과는 posts/{post.id}/createdatt/ 가 됩니다.
  • 따라서 아래와 같이 url을 추가해줍니다.

createdatt url 추가

urlpatterns = [
    path('<int:post_id>/createdatt/', views.p_createdatt, name="createdatt"),
]

def p_createdatt 제작

  • 아래와 같이 p_createdatt 함수를 작성합니다.
  • p_createdatt는 과거 p_create와 거의 유사합니다. 하지만 매우 큰 차이점이 존재합니다.
  • 이전에 포스트의 경우 postform의 내부에 2가지변수들만 받기 때문에 작성자와 글내용만 채워주면 새 포스트를 추가할 수 있었습니다.
  • 하지만 댓글의 경우 작성자, 글내용 뿐만아니로 해당 포스트에 대한 ForeignKey를 지정해주어야합니다.
  • 하지만 이전에 작성한 dattform의 경우 ForeignKey를 변수로 받지 않았습니다.
  • 따라서 댓글을 작성할 때 자동으로 해당 게시물에 대한 id를 ForeignKey로 받을 수 있어야합니다.
  • 아래 코드를 보시면 post_id가 pk인 post를 먼저 찾아주고 ForeignKey인 author_text에 지정해줍니다.
  • 그후 작성한 dattform의 instance를 author_text로 지정하여 ForeignKey를 설정합니다.
  • 마지막으로 제대로 된값을 입력하였다면 저장해주고 datt.html로 복귀해줍니다.
  • 아래 createdatt.html의 코드를 보시면 이전의 create.html과 거의 같고 datt_form을 작성해주는 것만이 다릅니다.
def p_createdatt(request,post_id):
    if request.method == 'POST':
        post = Post.objects.get(pk=post_id)
        author_text= Datt(author_text=post)
        datt_form=DattForm(request.POST,instance=author_text)# POST방식으로 전송된것을 싹다 사용해서 객체로 만들어라!
        if datt_form.is_valid(): # 제대로 된 값을 입력하였다면
            datt_form.save()

            post = get_object_or_404(Post, pk=post_id)
            datt = post.datt_set.all
            post_form = PostForm(instance=post)
            context={'post_form': post_form, 'datt':datt, "post":post}
            return render(request, 'datt.html', context) #다시 돌아간다!

    # GET 방식으로 호출될때 => form action을 사용하지않는 거의 모든 방식
    else:
        datt_form=DattForm() #PostForm의 인자가 ()처럼 비어있기 때문에 비어있는 형태로 출력된다.
        return render(request, 'createdatt.html',{'datt_form':datt_form})

댓글 삭제 기능 추가

  • 댓글을 작성하는 기능을 추가하였습니다. 이제는 작성한 댓글을 버튼하나로 지울수있는 기능을 추가해보도록합시다.
  • 앞에서 작성한 datt.html 을 보면 url 'posts:deletedatt' post.id datts.id으로 deletedatt의 url이 되어있는것을 확인할 수 있습니다.
  • 이러한 url의 결과는 posts/{post.id}/datt/{datt.id}/deletedatt 가 됩니다.
  • 따라서 아래와 같이 url을 추가해줍니다.

deletedatt url 추가

urlpatterns = [
    path('<int:post_id>/datt/<int:datts_id>/deletedatt/', views.p_deletedatt, name="deletedatt"),
]

def p_deletedatt 제작

  • 아래와 같이 p_deletedatt 함수를 작성합니다.
  • p_deletedatt 함수는 매우 간단합니다. 먼저 포스트들 중 앞에서 받은 datt_id가 pk인 댓글을 찾아줍니다.
  • 그리고 찾은것을 삭제해주고 다시 datt.html로 돌아갑니다.
  • 나머지 코드들은 datt.html로 복귀하였을 때 필요한 것들을 띄워주기위해 추가된것들입니다.
def p_deletedatt(request,post_id,datts_id):
    datt = Datt.objects.get(id=datts_id)
    datt.delete()

    post = get_object_or_404(Post, pk=post_id)
    datt = post.datt_set.all
    post_form = PostForm(instance=post)
    context = {'post_form': post_form, 'datt': datt, "post": post}
    return render(request, 'datt.html', context)

수정 기능 추가

  • 댓글을 삭제하는 기능을 추가하였습니다. 이제는 작성한 댓글을 수정할 수 있는 기능을 추가해보도록합시다.
  • 앞에서 작성한 datt.html 을 보면 url 'posts:updatedatt' post.id datts.id으로 updatedatt의 url이 되어있는것을 확인할 수 있습니다.
  • 이러한 url의 결과는 posts/{post.id}/datt/{datt.id}/updatedatt 가 됩니다.
  • 따라서 아래와 같이 url을 추가해줍니다.

updatedatt url 추가

urlpatterns = [
    path('<int:post_id>/datt/<int:datts_id>/updatedatt/', views.p_updatedatt, name="updatedatt"),
]

def p_updatedatt 제작

  • 아래와 같이 p_updatedatt 함수를 작성합니다.
  • p_updatedatt 함수는 앞에서 제작하였던 p_createdatt와 매우 유사합니다.
  • 다른점은 p_createdatt의 경우 처음 get방식으로 왔을 때 빈 dattform을 호출하지만 p_updatedatt의 경우 datt_id를 설정하여 기존의 댓글을 지정하여 호출하는 것입니다.
  • 그리고 찾은것을 수정해주고 저장해줍니다.
  • 저장하였다면 datt.html로 돌아갑니다.
def p_updatedatt(request,post_id,datts_id):
    post= get_object_or_404(Post,pk=post_id)
    datt= get_object_or_404(Datt,pk=datts_id)

    if request.method == 'POST':
        datt_form = DattForm(request.POST, instance=datt)

        if datt_form.is_valid():
            datt_form.save()

            post = get_object_or_404(Post, pk=post_id)
            datt = post.datt_set.all
            post_form = PostForm(instance=post)
            context = {'post_form': post_form, 'datt': datt, "post": post}
            return render(request, 'datt.html', context)

    else:
        datt_form = DattForm(instance=datt)

    return render(request, 'createdatt.html', {'datt_form': datt_form})
