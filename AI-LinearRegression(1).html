기본부터 시작해보자

기본적으로 필수로 사용되는 module은 pandas, numpy, tensorflow등이 있습니다.

import numpy as np
import pandas as pd
import tensorflow as tf
import matplotlib.pyplot as plt

1. Raw Data Loading (데이터 호출)

df=pd.read_csv('./data/data.csv')
display(df)
  • pandas 모듈의 read_csv를 활용하여 csv파일을 읽어옵니다.
  • 그 후 display를 통해 잘읽었는지 확인해봅시다.

2. Data Preprocessing(데이터전처리)

데이터 전처리는 2개의 종류가 있습니다.

1번은 결측치 수정

2번은 이상치 수정

2.1 결측치 수정

결측치는 측정과정에서 문제가 발생하여 데이터 내부에 구멍이 생긴것입니다. 이러한경우 해결책은 2가지입니다.

결측치를 다른 내용으로 매꾸거나 결측치 행을 제거하는 것입니다.

현재의 단계에서는 결측치를 제거하는것으로 하겠습니다.

df.isnull().sum() # pandas의 isnull함수를 사용하여 결측치의 수를 측정합니다.
df=df.dropna(how='any') # data 결측치 처리
2.2 이상치 수정

이상치는 측정과정에서 문제가 발생하여 데이터가 이상한 값들로 채워진 것입니다.

결측치와 마찬가지로 이러한경우 해결책은 2가지입니다.

이상치를 다른 내용으로 매꾸거나 이상치 행을 제거하는 것입니다.

현재의 단계에서는 이상치를 제거하는것으로 하겠습니다.

# 이상치의 존재를 확인하기 위하여 plt 함수를 활용하겠습니다. 
# 이러한 함수를 사용하면 이상치의 존재를 파악할 수 있습니다.
fig=plt.figure()
fig_gre=fig.add_subplot(1,4,1)
fig_admit=fig.add_subplot(1,4,2)
fig_gpa=fig.add_subplot(1,4,3)
fig_rank=fig.add_subplot(1,4,4)
fig_gre.boxplot(df['gre'])
fig_admit.boxplot(df['admit'])
fig_rank.boxplot(df['rank'])
fig_gpa.boxplot(df['gpa'])
fig.tight_layout()
plt.show()

# 이상치를 제거하자!
from scipy import stats # 이상치를 제거하기위하여 scipy 모듈을 추가합니다.

zscore_threshold=2.0 # zscore_threshold를 설정해줍니다. 2는 약 97%

for col in df.columns:
    outlier=df[col][np.abs(stats.zscore(df[col]))>zscore_threshold] 
    # 각 컬럼의 값들의 표준 이산확률분포 값들이 앞의 threshold을 넘는지 확인하여 outlier로 설정
    df=df.loc[~df[col].isin(outlier)] # threshold를 넘는값들을 제거!

3. Training Data Set

이상치와 결측치를 제거하였다면 변수 데이터인 x_data와 정답 데이터인 t_data로 나눠서 교육데이터를 설정해주겠습니다.

x_data= df.drop('admit',axis=1,inplace=False).values # 정답행인 admit을 제외한 것을 x_data로 설정
t_data= df['admit'].values.reshape(-1,1) # 정답행인 admit을 설정

4. normalization(정규화)

정규화란 데이터의 크기차 때문에 발생하는 문제를 제거하기위하여 데이터의 범위를 균일하게 만들어주는것을 의미합니다.

from sklearn.preprocessing import MinMaxScaler 
# 정규화를 진행하기 위하여 sklearn의 MinMaxScaler를 추가합니다.
scaler_x=MinMaxScaler()
scaler_x.fit(x_data) # x_data의 정규화를 진행합니다.
norm_x_data=scaler_x.transform(x_data)

여기까지가 python, tensorflow, sklearn을 활용하기전 공통으로 진행하는 과정입니다.

이제 sklearn을 사용하여 교육을 진행하겠습니다.

5. sklearn을 사용한 교육 진행

sklearn의 경우 다른 과정을 거치지않고 몇개의 명령어만 수행하면 교육을 진행할 수 있습니다.

from sklearn import linear_model # sklearn의 선형 모델을 추가

model=linear_model.LogisticRegression()
model.fit(x_data,t_data.ravel()) # 이것으로 교육을 완성

my_score=np.array([[600,3.8,1]]) # 교육된 모델을 사용해서 예측을 진행
predict_val=model.predict(my_score) # 0 or 1 교육 결과
predict_proba=model.predict_proba(my_score) # 교육된 결과의 확률
print('공부시간 {}, 확률{} {}'.format(my_score,predict_val,predict_proba))

다음 포스트에서는 tensorflow와 python은 다음 포스트에서 진행하겠습니다.
