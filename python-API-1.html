Python API 비 로그인식 크롤링

이번 포스트에서는 파이썬에서 사용했던 API들을 정리보겠습니다.

1. 네이버 API

  • 네이버 API를 사용하기 위해서는 아래의 싸이트에서 어플리케이션을 등록하고 Client ID와 Client Secret를 제공받아야 합니다.

네이버 developers

  • 네이버 API는 다양한 서비스를 제공합니다.

    • 로그인식

      1. 네이버 아이디로 로그인: 별도의 아이디와 비밀번호 없이 네이버 아이디로 간편하게 외부 서비스에 로그인할 수 있게 하는 API입니다.
      2. 카페: 외부 서비스에서 네이버 카페에 가입하거나 게시글을 등록할 수 있게 하는 API입니다.
      3. 캘린더: 외부 서비스에 네이버 캘린더에 일정을 등록할 수 있게 하는 API입니다.

    • 비로그인식

      1. 검색: 네이버 검색 결과를 뉴스, 백과사전, 블로그, 쇼핑, 영화, 웹 문서, 전문정보, 지식iN, 책, 카페글 등 분야별로 볼 수 있는 API입니다. 그 외에 지역 검색 결과와 성인 검색어 판별 기능, 오타 변환 기능을 제공합니다.
      2. 공유하기: 콘텐츠를 네이버 블로그, 네이버 카페, PHOLAR에 공유할 수 있게 하는 API입니다.
      3. 단축URL: 원본 URL을 http://me2.do/example과 같은 형태의 짧은 URL로 반환받을 수 있는 API입니다.
      4. 지도: 웹 서비스나 애플리케이션에 네이버 지도를 활용할 수 있게 하는 API입니다.
      5. 캡차(이미지): 네이버 서비스에서 사용하는 이미지 캡차 기능을 외부 서비스에 사용할 수 있게 하는 API입니다.
      6. Clova Face Recognition: 입력된 사진 이미지 속의 얼굴을 인식하거나 얼굴 감지를 이용한 애플리케이션을 만들 수 있게 하는 API입니다.
      7. Papago 번역: 인공 신경망 기술 기반의 기계 번역 결과를 반환하는 API입니다.

  • 이러한 API들 중 이번 포스트에서는 네이버 검색들 중 뉴스 검색을 예제로서 설명하겠습니다.

import requests
from urllib.parse import quote
import re

news=input("검색어를 입력하세요:")

client_id = {Client ID}
client_secret = {Client Secret}
headers = {'X-Naver-Client-Id':client_id, 'X-Naver-Client-Secret':client_secret}


url_base="https://openapi.naver.com/v1/search/news.json?query="
keyword = quote(news)
url_middle="$&start="
keyword_number='1'

url = url_base + keyword + url_middle + keyword_number

result = requests.get(url,headers = headers).json() #결과값을 변수로 지정

for z in range(0,5):
    zz=re.sub('[<b></b>&quot;]','',result['items'][z]['title'])
    print("{}번 기사:".format(z+1)+zz)
    print("{}번 URL:".format(z+1)+result['items'][z]['link'])
  • 위의 코드에 발급받은 Client ID와 Client Secret을 기입한 후 실행시키면 검색어를 입력할 수 있습니다.
  • 검색어를 입력하면 해당 검색어의 네이버 뉴스의 최신순으로 5개의 결과 값이 출력되는 것을 확인하실 수 있습니다.

  • 검색 API의 경우들도 key값들을 제외하고 위와 유사하게 사용하시면 됩니다.

2. 카카오 API

  • 카카오 API를 사용하기 위해서는 아래의 싸이트에서 어플리케이션을 등록하고 REST API를 제공받아야 합니다.

kakao developers

  • 카카오 API는 네이버 API와 마찬가지로 많은 기능을 제공합니다.
  • 하지만 결과적으로 2가지로 나뉩니다.
    • 카카오톡 연동
    • 카카오톡 비연동

카카오톡 비연동

  • 카카오톡 비연동 중에서는 다음 검색들 중 웹문서 검색을 예제로서 설명하겠습니다.
import requests
import json

news=input("검색어를 입력하세요:")

headers = {'Authorization':"KakaoAK {REST API}"}
query = {"query":news}

urls= "https://dapi.kakao.com/v2/search/web"

result = requests.get(urls,headers = headers, params=query) #결과값을 변수로 지정

print(json.loads(result.text))
  • 위의 코드에 발급받은 REST API을 기입한 후 실행시키면 검색어를 입력할 수 있습니다.
  • 이때 중요한것은 kakaoAK를 지우면 안된다는 것입니다.
  • 검색어를 입력하면 해당 검색어의 다음 웹문서가 10개가 JSON형태로 출력됩니다.
  • 하지만 이 포스트에서는 결과물이 너무 길기 때문에 중간에 잘랐습니다.
